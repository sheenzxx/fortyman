mysql json 类型

发表于 2023-04-04 更新于 2023-04-06 分类于 mysql

JSON 类型的存储跟LONGBLOB, LONGTEXT 一样 存储长度:L + 4 bytes, where L < 2 的32次方
JSON 的传输限制于 max_allowed_package 的大小，可以通过 JSON_STORAGE_SIZE() 函数来确定json 文档的大小
JSON 类型并不能直接创建索引，而是由计算列支持
